# Lisp 基本语法
表达式（算术运算表达式）
<pre>
> (+ 1 2)
< 3
> (+ 1 2 3 4)
< 10
> (+ (* 3 3) (* 4 4))
< 25
</pre>

较长的表达式（组合的表达式）
<pre>
(+ (* 3 (+ (* 2 4) (+ 3 5))) (+ (- 10 7) 6))

(+ (* 3
        (+ (* 2 4) 
            (+ 3 5)))
    (+ (- 10 7) 
        6))
</pre>

命名（也叫定义，不叫赋值）
<pre>
< (define size 5)
> 
< (* size 2)
> 10
</pre>

复合过程（函数定义）
<pre>
< (define (add a b) (+ a b))
< (add 1 2)
> 3
< (define (square x) (* x x))
< (square 4)
> 16
</pre>
